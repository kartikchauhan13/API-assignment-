Shopping Cart Products API

HTTP Methods used in the API :

  • POST --> Creating a new product
  • GET --> getting the info of all products
  • PATCH --> updating a product
  • DELETE --> deleting a product

Creating a product

  • Method : POST
  • URL : http://kcmail823.pythonanywhere.com/api/cart-items/

Getting info of products

  • Method : GET
  • URL : http://kcmail823.pythonanywhere.com/api/cart-items/

Getting info of a single product

  • Method : GET
  • URL : http://kcmail823.pythonanywhere.com/api/cart-items/id/
  • Example product id =5 : http://kcmail823.pythonanywhere.com/api/cart-items/5/

Updating a product

  • Method : PATCH
  • URL : http://kcmail823.pythonanywhere.com/api/cart-items/id/
  • Example for product id =5 : http://kcmail823.pythonanywhere.com/api/cart-items/5/

Deleting a product

  • Method : DELETE
  • URL : http://kcmail823.pythonanywhere.com/api/cart-items/id/
  • Example for product id =5 : http://kcmail823.pythonanywhere.com/api/cart-items/5/
